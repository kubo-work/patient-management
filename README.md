# patient-management
## 概要
私が毎週お世話になっている整骨院様向けに作成した患者管理アプリです。
実際にこういうのが欲しいと言われた機能をヒアリングして作成しました。
※ただしまだ途中です。


## 使用した言語やライブラリ ※一部です
- フロントエンド
    - NextJS
    - React
    - mantine ui
- バックエンド
    - NodeJS
    - express
    - prisma

## デプロイ先
- フロントエンド : vercel
- バックエンド : render

## ログインURL
